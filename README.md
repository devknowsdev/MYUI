## MIR docs

The MIR handover material lives under `docs/mir/` with stable filenames:

- `docs/mir/handover.md` — top-level handover and navigation
- `docs/mir/lexicon-schema.md` — term model, metadata model, graph structure
- `docs/mir/import-export.md` — verification workflow plus CSV/JSON import-export shapes
- `docs/mir/open-questions.md` — unresolved decisions and next actions

## Documentation convention

When adding or moving important docs:

1. keep filenames stable
2. update `docs/INDEX.md`
3. add links from the nearest parent index
4. prefer exact paths in PRs and chats when requesting review
